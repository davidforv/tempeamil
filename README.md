## 🎯 为什么我们需要 TMPL Email？

在数字时代，下载一份报告、注册一个小众论坛、或者白嫖几天软件会员，都逼你交出真实邮箱。随之而来的就是没完没了的营销广告、垃圾账单，甚至遭遇数据库泄露导致的撞库风险。

**TMPL Email 就是你的数字防弹衣。** 

我们提供秒级生成、即用即走、阅后即焚的匿名信箱，把垃圾留给过去，把清爽留给你的真实收件箱。

## ✨ 核心功能与降维优势

*   ⚡ **秒级无感生成**：打开网页 `0.1秒` 内自动分配全新邮箱地址，无需刷新，实时接收验证码、激活链接及附件。
*   🚫 **真正零门槛**：免注册、免密码、无繁琐弹窗广告、无任何付费套路，给用户最纯粹的极简体验。
*   🛡️ **硬核防封检测**：底层优化邮箱域名，完美绕过绝大多数国内外网站的“临时邮箱屏蔽检测”，确保验证码秒级送达。
*   🔒 **极致隐私审计**：前端不引入任何第三方利益链的追踪脚本（Analytics/Tracking），不收集用户指纹。

本项目作为 `https://tmpl.email` 的官方前端，具备极高的工程参考价值：
